Create a user named Lynda Herman and assign the Read Only Clone profile to her

问题1:

Developer Edition里没有找到到ReadOnly的profile，有一个MinimumAccess-Salesforce的Profile，不知道是不是一样的？

原因:

版本问题？

解决:

启动Playgroud后，在Profile的设定里就能找到ReadOnly的profile

问题2:

新建用户以后在Profile的下拉框里没有ReadOnlyClone的选项。

原因:

ReadOnlyClone的profile属于salesforce的License，但是在company information里发现salesforce的剩余license已经是0

解决:

找到一个使用salesforce的license的用户，把他的license暂时设置成其他license，等做完挑战任务再重设回来。
